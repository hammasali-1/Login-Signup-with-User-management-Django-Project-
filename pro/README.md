# Login-Signup with User Management Django Project

## Overview
This project is a complete **user authentication and management system** built using **Django**. It allows users to **sign up, log in, update their profile, and manage customer records**. The project features a **modern, responsive design** with **light/dark theme support**, subtle **animations**, and a clean user interface.  

It also implements a **role-based access control system** to manage user permissions effectively:  

- **Admin**: Full control over the system, including updating, and deleting customer records.  
- **Manager**: Can read, and update customer records, but **cannot delete users**.  
- **User**: Can only view data; cannot update or delete any customer records. 

---

## Features
- **User Signup and Login** with secure password handling  
- **Profile update** functionality  
- **CRUD operations** for managing customer data  
- **Role-based access control** for Admin, Manager, and User  
- Modern **responsive UI** using Bootstrap 5  
- **Light and Dark mode toggle** (persistent across sessions)  
- Subtle **animations** for enhanced user experience  
- Fully compatible with **Django backend**  

---

## Technology Stack
- **Backend**: Django  
- **Frontend**: HTML, CSS, Bootstrap 5  
- **Database**: SQLite (default for Django, can be changed)
